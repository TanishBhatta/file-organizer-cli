# file-organizer-cli
A terminal-based Python CLI that automatically sorts files into subfolders by extension. Built with os and shutil - no dependencies.

**What it does** -
This script sorts files in its current directory into specific subfolders based on their file extensions to keep the file system tidy.

**How to run it** -
python file_organizer.py

**Supported extensions** -
* Video: .mp4, .mkv, .mov
* Audio: .mp3, .wav, .aac
* Images: .jpg, .jpeg, .png, .gif
* Documents: .pdf, .docx, .txt, .xlsx
* Archives: .zip, .rar, .7z
* Code: .py, .js, .html, .css
* Executables: .exe, .msi, .dmg
* Fonts: .ttf, .otf
* Data: .json, .csv, .xml
* Presentations: .pptx, .key
